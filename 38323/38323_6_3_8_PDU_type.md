### 6.3.8 PDU type

Length: 3 bits

This field indicates the type of control information included in the
corresponding PDCP Control PDU.

Table 6.3.8-1: PDU type

  ---------------- ------------------------------------------------------
  Bit              Description

  000              PDCP status report

  001              Interspersed ROHC feedback

  010              EHC feedback

  011              UDC feedback

  100              PDCP SN gap report

  101-111          Reserved
  ---------------- ------------------------------------------------------
